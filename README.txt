# Sistema de Control Personal — versión NeoTech

## Cómo usar
1. Descomprime el ZIP.
2. Abre `index.html` en Chrome o Edge.
3. Presiona **Activar notificaciones** y acepta.
4. Deja el navegador abierto o instala la app desde el navegador.

## Nota importante
Las notificaciones son locales del navegador. Para que salten, el sitio debe estar abierto o instalado como app.
